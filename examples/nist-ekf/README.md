# NIST EKF Demo

The NIST EKF demo lives in its own repository:

https://github.com/Enterprise-knowledge/nist-ekf

It demonstrates Enterprise Knowledge Format applied to public governance material from:

- NIST Cybersecurity Framework 2.0
- NIST AI Risk Management Framework 1.0
- NIST SP 800-53 Rev. 5

The demo includes nested EKF bundles, generated markdown artifacts, source provenance, typed relationships, graph JSON, and a static HTML knowledge graph.

![NIST EKF knowledge graph demo](../../assets/nist-ekf-knowledge-graph.png)

The full demo is intentionally kept outside this standard repository because it contains large source documents and generated artifacts. This keeps the standard repo small while still giving launch readers a concrete example to inspect.
