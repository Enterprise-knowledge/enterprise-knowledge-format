# Changelog

All notable changes to Enterprise Knowledge Format will be documented here.

This project follows semantic versioning once the standard reaches a stable release. Until then, `0.x` releases may refine schema, conformance, and examples based on implementation feedback.

## 0.1.0 - Draft

- Added the initial EKF specification.
- Added the `ekf-bootstrap` skill for creating starter EKF bundles.
- Added validation tooling for checked authoring-conformance rules.
- Added graph parsing support for `related` frontmatter.
- Added a reusable static HTML graph template.
- Added MarkItDown guidance for source-to-knowledge ingestion.
